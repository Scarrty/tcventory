App Name: TCventory

Ziel:
Eine App zur Inventarverwaltung von  TCG-Karten und Sealed Produkten.

Hauptfunktionen:
- Inventarisierung
- Katalogisierung
- Tracking von Einkauf, aktuellem Wert, Verkauf inkl. Versand und Gebühren
- Tracking von Zuständen, Grading jeder einzelnen Karte
- Audit-sicheres Tracking von Veränderungen am Bestand

Plattform:
Web App + Responsive Layout for Mobile
